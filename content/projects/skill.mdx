# 自己紹介

私はフロントエンドエンジニアとして、HTML、CSS、JavaScriptなどのWeb技術に長けています。ReactやNext.js、Vue.jsなどのフロントエンドフレームワークを使った開発経験があり、コンポーネント志向の考え方を身に付けています。

バックエンドでは、Node.jsやPHPなどの言語やフレームワークを使った経験があります。REST APIの設計やデータベース操作ができます。

GSAP、Three.js、Matter.jsなどのJavaScriptライブラリを使い、インタラクティブで洗練されたUI/UXを提供できます。

頻繁に更新される最新のWeb技術に対応できるよう、日々学習に励んでいます。例えば、最近はAstroやSvelteKitなどの新しいフレームワークを使ったサイト構築を行っています。

ツール面では、Gitによるバージョン管理、NPMによるパッケージ管理、webpackによるモジュールバンドルができます。開発効率と品質の向上に力を入れています。

将来的には、テスト駆動開発(TDD)を取り入れたり、サーバーサイドレンダリング(SSR)の改善に取り組む予定です。常により良いコードとサイトを提供できるよう尽力し続けます。

## スキルと経験

### 言語
```md
- HTML/CSS
- JavaScript/TypeScript
- PHP
```
私はHTML/CSSを用いたマークアップとスタイリングに堪能です。SEOとアクセシビリティに配慮した実装ができます。JavaScript/TypeScriptでのインタラクティブなUI制御もできます。また、PHPを使ったサーバーサイドの処理経験もあります。

### データベース
```md
- MySQL 
- PostgreSQL
```
Webアプリケーションでよく使用されるRDBMSです。SQLによるデータ操作ができます。正規化、インデックスの設計等も可能です。

### インフラ
```md
- GCP
- AWS 
- Docker
```
Google Cloud Platform、Amazon Web Services等のクラウド環境でのアプリケーションデプロイができます。Dockerコンテナを使った開発/運用環境の構築経験もあります。

### 情報共有ツール
```md
- Slack
- ZOOM
- Chatwork 
- Backlog
```
チーム開発におけるコミュニケーション能力があります。これらのツールを使ったオンライン会議、文書共有、タスク管理ができます。

### ソースコード管理
```md
- GitHub
- GitLab
- Bitbucket
```
GitHub、GitLab、Bitbucketなどのバージョン管理ツールを使いこなせます。コードレビューやCI/CDパイプラインの構築経験もあります。

### フレームワーク
```md
- WordPress
- React.js
- Next.js
- Astro
```
これらのフレームワークを使用した実務経験があります。品質の高いサイト構築・運用ができます。

### デザインスキル 
```md
- Photoshop
- Illustrator
- Visual Studio Code
- Figma
- Canva
```
PhotoshopやIllustratorを使った画像編集スキルがあります。VSCode等のIDEを使った実装能力も兼ね備えています。モダンなFigmaやCanvaも使いこなせます。
